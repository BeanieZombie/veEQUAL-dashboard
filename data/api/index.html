üîß Usage Information

Rate Limits

No rate limits currently enforced. Data is updated daily via GitHub Actions.

Data Freshness

All endpoints return cached data that is refreshed daily at approximately 00:00 UTC.

CORS

Cross-origin requests are supported for web applications.

Authentication

No authentication required. All endpoints are publicly accessible.
